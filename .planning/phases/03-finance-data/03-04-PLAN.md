**B. `lauos/src/components/finance/category-form.tsx` — `'use client'`**

```typescript

  mode: 'create' | 'edit'
  initialValues?: { id: string; name: string; icon: string; color: string }
  onSuccess: () => void
}
```

Form fields:
1. **Name** (shadcn Input, required, maxLength 60)
2. **Icon** (shadcn Input, required, maxLength 10, placeholder "🛒" — user types or pastes any emoji)

   - A row of 8 preset color swatches (small clickable squares: #6366f1, #22c55e, #f59e0b, #ef4444, #3b82f6, #a855f7, #14b8a6, #f97316). Clicking a swatch sets the color value.

4. **Live preview**: Show a CategoryBadge at the bottom of the form using the current name/icon/color so the user can see how it will look.

On submit:
- `mode === 'create'` → `createCategoryAction({ name, icon, color })`
- `mode === 'edit'` → `updateCategoryAction({ id, name, icon, color })`
- Show inline error from action response
- Call `onSuccess()` on success
- `useTransition` for loading state
