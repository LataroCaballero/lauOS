**Step A — Create lauos/src/lib/actions/insights.ts:**

Follow the exact pattern from 04-RESEARCH.md code examples. Implement two server actions:

1. `getMonthlySummaryAction(params: { year: number; month: number })`:
   - Returns `{ summaries: MonthlySummary[] } | { error: string }`
   - Fetches transactions filtered by `user`, `date >= start && date <= end`, and `(type = "income" || type = "expense")` — transfers ALWAYS excluded
   - Groups by account currency using a currency map from accounts collection
   - Month boundaries: start = first day 00:00:00.000Z, end = last day 23:59:59.999Z (use `new Date(year, month, 0).getDate()` for last day)
   - Returns separate `MonthlySummary` entries per currency (only currencies with data)

2. `getCategoryDistributionAction(params: { year: number; month: number; type: 'income' | 'expense'; currency: 'ARS' | 'USD' })`:
   - Returns `{ slices: CategorySlice[] } | { error: string }`
   - Fetches transactions of given type for month, for accounts of given currency, expands `category`
   - Groups by category, sums `amount_centavos`
   - Transactions with no category → `{ categoryId: null, categoryName: 'Sin categoría', categoryColor: '#9ca3af' }`
   - Returns slices sorted by `amountCentavos` descending

Use `createServerClient` from `@/lib/pocketbase-server`. Guard with `if (!pb.authStore.isValid) return { error: 'Not authenticated' }`. Wrap in try/catch returning `{ error: e instanceof Error ? e.message : 'Failed' }`.

Export types: `MonthlySummary`, `CategorySlice`.

**Step B — Create lauos/src/app/(protected)/finance/insights/insights-client.tsx (Client island):**

Props:
```typescript
type InsightsClientProps = {
  initialSummaries: MonthlySummary[]
  initialSlices: CategorySlice[]
  accounts: AccountWithBalance[]
  initialYear: number
  initialMonth: number
}
```

State:
- `year: number`, `month: number` — currently selected month (initialized from props)
- `summaries: MonthlySummary[]` — refreshed on month change
- `slices: CategorySlice[]` — refreshed on month change
- `activeTab: 'expense' | 'income'` — Gastos / Ingresos tab
- `loading: boolean`

Month selector: a `<Select>` (from `@/components/ui/select`) showing month names in Spanish + year. Build a list of the last 24 months (current month back to 24 months ago) as options. On change: set year+month state, call `getMonthlySummaryAction` and `getCategoryDistributionAction` in parallel, update summaries+slices.

Layout (top to bottom):
1. **Month selector** — full-width or right-aligned, shows "Marzo 2026" format
2. **Summary cards row** — map `summaries` to `<MonthlySummaryCard>` (one per currency). Show "Sin datos para este mes" if empty.
3. **Category chart section** — heading "Distribución por categoría", tab buttons (Gastos / Ingresos) using `<Button variant="outline">` (active tab gets `variant="default"`), then `<CategoryDonutChart>` with current slices. On tab change: fetch new slices with updated type, keep same month.
4. **Category legend** — simple list below chart: for each slice, show `<CategoryBadge>` (from `@/components/finance/category-badge`) + `fromCentavos` amount + percentage of total. Only show if slices exist.

When `loading === true`, show a subtle opacity-50 on the content sections (not full skeleton — keep it simple).

Re-fetch pattern on month change:
```typescript
async function handleMonthChange(newYear: number, newMonth: number) {
  setYear(newYear)
  setMonth(newMonth)
  setLoading(true)
  const [summaryResult, slicesResult] = await Promise.all([
    getMonthlySummaryAction({ year: newYear, month: newMonth }),
    getCategoryDistributionAction({ year: newYear, month: newMonth, type: activeTab, currency: 'ARS' }), // default ARS; could show both
  ])
  if ('summaries' in summaryResult) setSummaries(summaryResult.summaries)
  if ('slices' in slicesResult) setSlices(slicesResult.slices)
  setLoading(false)
}
```

Note: The category chart shows one currency at a time. Default to ARS. If only USD accounts exist, default to USD. Use the `accounts` prop to determine which currencies are present.
