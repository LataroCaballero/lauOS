Create `lauos/src/lib/actions/accounts.ts` with the following four exported Server Actions. All follow the established pattern: `'use server'` directive, `createServerClient()`, `isValid` guard, explicit `user` field on creates, `revalidatePath('/finance')` after mutations.

**createAccountAction:**
```typescript
'use server'
import { createServerClient } from '@/lib/pocketbase-server'
import { revalidatePath } from 'next/cache'

export async function createAccountAction(data: {
  name: string
  currency: 'ARS' | 'USD'
}): Promise<{ id?: string; error?: string }> {
  const pb = await createServerClient()
  if (!pb.authStore.isValid) return { error: 'Not authenticated' }
  try {
    const record = await pb.collection('accounts').create({
      user: pb.authStore.record?.id,
      name: data.name.trim(),
      currency: data.currency,
      archived: false,
    })
    revalidatePath('/finance')
    return { id: record.id }
  } catch (e: unknown) {
    return { error: e instanceof Error ? e.message : 'Failed to create account' }
  }
}
```

**updateAccountAction:** Accepts `{ id, name }` — currency cannot be changed after creation (schema is immutable once data exists per roadmap decision).

**archiveAccountAction:** Accepts `{ id }` — sets `archived: true`. Does NOT delete the record or its transactions.

**getAccountsWithBalancesAction:** Fetches all non-archived accounts, then fetches ALL user transactions in a single query (fields: account, type, amount_centavos), builds a balance map using the formula above, and returns:
```typescript
Array<{ id: string; name: string; currency: 'ARS' | 'USD'; balanceCentavos: number }>
```
Also compute and return:
```typescript
patrimonySummary: { totalArsCentavos: number; totalUsdCentavos: number }
```
by filtering the returned accounts by currency and summing their balanceCentavos.

Filter accounts with `archived = false` in the PocketBase query. Sort by `created` ascending.

Do NOT allow currency updates — if an `updateAccountAction` is called with a currency change, ignore it (only name is editable).
